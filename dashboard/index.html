系統敘事

這不是一份靜態文件，而是一個活的生態系統。從最初的一台筆電 + Claude CLI，到現在 NB/WSL2 + Cloud MCP 雙節點、Mac Mini M2 獨立 Discord 節點、Contains Studio 六部門 AI Agent，整個系統經歷了數個月的演化與迭代。

核心哲學只有一條：「把任務說清楚，系統幫你找技能。」每個技能都是一個獨立的執行單位（Skill），像樂高積木一樣可以組合、替換、升級。當任務來了，系統會自動比對 122+ 個技能的功能標籤，找到最適合的組合。
